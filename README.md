# Tactical RPG

A tactical RPG built with Godot 4 (GDScript), featuring Tetris-style grid inventory, gem modifier system, passive skill trees, and ATB turn-based combat.

## Quick Start

1. Open in Godot 4.6+
2. Run `scenes/main_menu/main_menu.tscn`
