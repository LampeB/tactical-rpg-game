# Item Catalog

All items in the game, grouped by type. Each item uses a single sprite across all rarities — rarity is shown via grid cell tint color.

Sprite sizes: 1 cell = 64x64 px. A 2x1 item = 128x64 px, etc.

## Sprite Files (assets/sprites/items/)

Each item has its own sprite file named `{item_base_id}.png` (e.g., `sword.png`, `katana.png`, `claymore.png`).
Rarity distinction is cell tint only — all rarities of an item share the same sprite.

Currently all sprites are placeholder copies. Edit each `.png` to give items unique art.

Naming convention: `{item_base_id}.png` where item_base_id matches the id without the rarity suffix.

**Standalone sprites** (not tied to a specific item):
- `gem.png` — generic gem icon
- `orb.png` — spatial rune icon
- `heart.png` — UI heart icon
- `Arrow01(32x32).png` — arrow projectile sprite
